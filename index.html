Q1

Foundation Phase

Months 1-3: Stability First
💼 Employment Goals
Week 1-2: Secure Blue Collar Job Warehouse (Amazon, UPS, FedEx), Construction, Restaurant (Dishwasher/Kitchen), Retail (Stocking), Cleaning Services
Target: $15-18/hour minimum Apply to 10 places daily until hired. Use: Indeed, Craigslist, walk-ins
📄 Documentation Goals
Week 1: OPT Application (if applicable) Complete I-765 form, gather documents, submit to USCIS
Weeks 2-12: Track Application Check status weekly, prepare for EAD card receipt (90-120 days typical)
🏠 Housing Goals
Week 2-3: Secure Apartment Options: Shared room ($400-700), Studio ($800-1200), Roommates (Craigslist, Facebook groups)
Budget: Max 30% of income Use: Zillow, Apartments.com, University housing boards, Craigslist
🎯 Foundation Skills
Month 2-3: Data Science Exploration (Evenings) Python basics (2 hours/week), Excel mastery, Understand how chemistry → data science
Free Resources Started Python: Codecademy free tier, Excel: YouTube (ExcelIsFun), Chemistry+DS: Research papers
💪 Health Foundation
Month 3: Gym Habit Started 3x per week, 30 minutes. Focus: Build routine, not perfection. Planet Fitness ($10/month)
Q2

Growth Phase

Months 4-6: Skill Building & Career Launch
🎓 Learning Roadmap
Month 4: Python & Statistics Foundation Python for Data Science (Coursera - Google/IBM), Pandas, NumPy, Basic stats refresher
Month 5: Data Analysis & Visualization Matplotlib, Seaborn, Tableau/Power BI, SQL fundamentals
Month 6: Chemistry-Specific Applications ChemInformatics, Spectral data analysis, Molecular modeling with Python
🛠️ Portfolio Projects
Project 1 (Month 4): Analytical Chemistry Dataset Analysis Analyze HPLC/GC-MS data, predict compound properties, visualize results
Project 2 (Month 5): Drug Discovery Dashboard Interactive Tableau dashboard showing molecular properties vs. activity
Project 3 (Month 6): ML Model for Chemical Properties Predict solubility/toxicity using sklearn, document on GitHub
💼 Corporate Job Hunt
Month 4-5: Resume & LinkedIn Overhaul Highlight: MSc Chemistry + Data skills, Portfolio links, OPT status clear
Month 5-6: Apply to Entry-Level Roles Titles: Junior Data Analyst, Lab Informatics Analyst, Chemical Data Scientist, QA Data Analyst
Target: 20 applications/week Companies: Pharma (Pfizer, Merck, AbbVie), Chemical (Dow, BASF, DuPont), Startups (BenchSci, Atomwise)
💪 Fitness Consistency
Months 4-6: Solidify Gym Routine 4x per week, 45 minutes. Add: Strength training program, Track progress
🌐 Network Building
Join Communities LinkedIn groups: ChemInformatics, Data Science, Attend 2 virtual meetups/month
